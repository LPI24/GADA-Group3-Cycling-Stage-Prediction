| Feature | Beschreibung | Datentyp |
|---|---|---|
| `distance` | Distanz der Etappe in Kilometern | numerisch |
| `vertical_meters` | Höhenmeter der Etappe | numerisch |
| `stage_nr` | Nummer der Etappe innerhalb der Rundfahrt | ganzzahlig / ordinal |
| `team_tier` | Vereinheitlichte Teamkategorie | kategorial |
| `age_at_race` | Alter des Fahrers am Etappentag | numerisch |
| `rider_bmi` | Body-Mass-Index des Fahrers | numerisch |
| `wind_stability_index` | Eigener Index zur Veränderung von Windgeschwindigkeit und Windrichtung zwischen Start und Ziel | numerisch |
| `weather_temp_mean` | Durchschnittliche Temperatur aus Start- und Zielwetter | numerisch |
| `weather_temp_trend` | Temperaturveränderung zwischen Start und Ziel | numerisch |
| `weather_rain_prob_mean` | Durchschnittliche Regenwahrscheinlichkeit aus Start und Ziel | numerisch |
| `weather_precipitation_mean` | Durchschnittlicher Niederschlag aus Start und Ziel | numerisch |
| `weather_humidity_mean` | Durchschnittliche Luftfeuchtigkeit aus Start und Ziel | numerisch |
| `gradient_final_km` | Steigung des finalen Kilometers | numerisch |
| `lag_rider_points_season` | Saisonpunkte des Fahrers aus dem Vorjahr | numerisch |
| `lag_rider_rank_season` | Saisonrang des Fahrers aus dem Vorjahr | numerisch |
| `lag_race_competitiveness_median` | Median der Vorjahres-Ranglistenpositionen des Fahrerfelds einer Etappe | numerisch |
| `lag_team_power_index` | Median der Vorjahres-Ranglistenpositionen der Fahrer eines Teams innerhalb einer Etappe | numerisch |
